# Falltuna Notes
这是秋天的金枪鱼稿件合集, 优先收录markdown文件格式的稿件, 其余格式(由于转化麻烦), 会很快加入, 尽情期待.
为什么需要这个网页呢? 因为很多平台(如微信公众平台、今日头条)并不支持LaTeX或者Mermaid Graph(尽管后者只在“递推数列”那几期出现过)
所以只能用截图的形式呈现给读者, 固然是不如原始状态的.
(尤其是头条的前几期……)
因此这个网站用于展示最原始(或是理想状态下)的falltuna笔记.
